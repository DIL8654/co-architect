# OWASP ASVS Summary

## Purpose

Application and API security guidance for systems with user access, integrations, or sensitive data.

## Key Guidance

- authentication and credential handling
- access control and authorization boundaries
- input validation and safe output handling
- cryptography and data protection
- error handling and safe logging
- API security and session protection

## Use When

Use when the system exposes APIs, supports external users, stores sensitive data, or has admin access paths.
