# UX Strategy

## Purpose

Define the current UX direction for CoArchitect AI.

## Current Scope

The MVP uses a workspace-centric enterprise SaaS shell with minimal visual noise and visible reasoning surfaces.

## Design Direction

- workspace-centric UX
- persistent left navigation shell
- table-first layout
- minimalistic design
- grounded agent visibility
- no organization-centric user flow
- demo flow optimized for evaluators

## Design Decisions

- navigation belongs in the shell
- working surfaces belong in the content panel
- findings should be inspectable without clutter
- diagrams and analysis should use most of the screen
