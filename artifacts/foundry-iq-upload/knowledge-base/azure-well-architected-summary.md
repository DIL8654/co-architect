# Azure Well-Architected Summary

Source:
Microsoft Azure Well-Architected Framework. Use the official Microsoft guidance as the canonical reference.

Summary:

- Reliability focuses on resilience, recovery, and continuity under failure.
- Security focuses on identity, access, data protection, network controls, and operational safeguards.
- Cost Optimization focuses on right-sizing, managed services, and financial awareness.
- Operational Excellence focuses on observability, automation, repeatability, and operational readiness.
- Performance Efficiency focuses on scaling, latency, throughput, and service fit.

CoArchitect use:

- Azure-centric architecture review
- cloud-native Azure service decisions
- reliability and operations guidance for Microsoft-hosted systems
