# ISO 25010 Summary

Source:
ISO/IEC 25010 quality model. Use the official standard as the canonical reference.

Summary:

- Functional suitability evaluates whether the system meets intended needs.
- Performance efficiency evaluates time behavior, resource use, and capacity.
- Compatibility evaluates coexistence and interoperability.
- Usability evaluates learnability, operability, and user effectiveness.
- Reliability evaluates maturity, availability, fault tolerance, and recoverability.
- Security evaluates confidentiality, integrity, authenticity, accountability, and non-repudiation.
- Maintainability evaluates modularity, analyzability, and modifiability.
- Portability evaluates adaptability and ease of transfer.

CoArchitect use:

- cloud-neutral product quality analysis
- quality attribute framing beyond infrastructure
- maintainability and system fitness reasoning
