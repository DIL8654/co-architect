# Prompts And Skills

## Purpose

Explain how AI-assisted development was part of the project delivery process.

## Current Scope

CoArchitect AI was built incrementally with GitHub Copilot and Codex assistance.

## AI-Assisted Development Usage

AI assistance was used for:

- product planning prompts
- architecture design prompts
- UX redesign prompts
- backend implementation prompts
- frontend implementation prompts
- reasoning-agent prompts
- documentation formalization prompts

## Development Narrative

Copilot and Codex acted as development agents for:

- planning
- scaffolding
- refactoring guidance
- verification loops
- documentation maintenance

Human direction controlled scope, product decisions, and trade-offs.

## Future Internal CoArchitect Skills

- Architecture Review Skill
- ADR Generation Skill
- Framework Selection Skill
- Trade-off Analysis Skill
- Documentation Formalization Skill
- Demo Preparation Skill

## Design Decisions

- use documentation as repository memory
- use prompts to keep product direction aligned across iterations
- separate AI-assisted development from AI behavior inside the product

## Future Enhancements

Future work can formalize reusable internal prompt templates and evaluation-ready development workflows.
