# Reasoning Agent Rubric

Evaluate each architecture review on:

- grounding quality
- framework selection quality
- reasoning transparency
- trade-off clarity
- recommendation usefulness
- safety and uncertainty handling
- ADR coherence
- evidence alignment

Suggested evaluator prompts:

- Did the selected frameworks match the architecture context?
- Were recommendations grounded in the chosen frameworks?
- Were trade-offs explicit and weighted against user priorities?
- Did the final score remain application-calculated?
- Did the ADR capture decision, alternatives, and consequences clearly?
