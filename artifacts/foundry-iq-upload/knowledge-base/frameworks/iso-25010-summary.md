# ISO 25010 Summary

## Purpose

Cloud-neutral guidance for overall software and system quality.

## Key Guidance

- functional suitability for business fit
- performance efficiency for capacity and response time
- reliability for recovery and stability
- security for confidentiality, integrity, and accountability
- maintainability for modularity and changeability
- portability for adaptability across environments

## Use When

Use when the architecture needs a broad quality review beyond cloud infrastructure decisions.
