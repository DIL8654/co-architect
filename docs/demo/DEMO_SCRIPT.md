# Demo Script

## Purpose

Give presenters a polished 5-minute walkthrough.

## Script

1. Open CoArchitect AI and show the Dashboard.
2. Point out Demo Architecture Journeys and explain that all data is synthetic.
3. Open Automated Video Analysis Platform.
4. Show the real diagram image and the Architecture Intelligence panel.
5. Open the latest analysis result.
6. Show selected frameworks, score, findings, recommendations, and trade-offs.
7. Open Agent Workflow and explain the multi-step reasoning chain.
8. Open Foundry IQ context and show framework, principle, trade-off, ADR, and workspace-memory grounding.
9. Open ADRs and show version history.
10. Return to Dashboard and compare the document processing score and findings.

## Talking Points

- CoArchitect AI is an AI architecture partner, not a generic chat UI.
- Azure AI Foundry is the integration path for expert reasoning.
- The Foundry IQ-style layer grounds recommendations in architecture knowledge and workspace memory.
- The Architecture Intelligence Score is calculated by application code.
- Seeded results are persisted snapshots, so the demo does not depend on live cloud calls.

## Future Enhancements

After the hackathon, richer collaboration, login/RBAC, Azure AI Search-backed retrieval, and PDF ADR export can be added.
