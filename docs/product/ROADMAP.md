# Roadmap

## Purpose

Describe the current and future delivery phases.

## Current Scope

The current MVP is a workspace-centric architecture reasoning platform for the hackathon.

## Phase 1 — Core MVP

- workspace-centric product shell
- diagrams and architecture descriptions
- analysis runs
- Architecture Intelligence Score
- findings, trade-offs, ADR generation

## Phase 2 — Reasoning Architecture

- framework selection
- context capture
- multi-stage orchestration
- visible agent trace

## Phase 3 — Foundry IQ-Style Grounding

- `IFoundryIqProvider`
- file-based knowledge retrieval
- composite retrieval with workspace memory
- explainable grounding references

## Phase 4 — Demo And Evaluation Readiness

- polished UX
- evaluator-focused documentation
- demo script and submission notes
- reliability and safety checks

## Phase 5 — Post-Hackathon Enhancements

- external identity and RBAC
- Azure AI Search-backed retrieval
- deeper collaboration and approvals
- deployment automation
