# Demo Script

## Purpose

Provide a polished 5-minute demo script for evaluators.

## Current Scope

Use the seeded synthetic demo journeys. The fastest path is Automated Video Analysis Platform because it includes a real diagram image, completed analysis, Foundry IQ context, agent trace, findings, trade-offs, and ADR version history.

See [../demo/DEMO_DATA_PLAN.md](../demo/DEMO_DATA_PLAN.md) and [../demo/DEMO_JOURNEYS.md](../demo/DEMO_JOURNEYS.md).

## Script

1. Open CoArchitect AI.
2. Show Demo Architecture Journeys on the dashboard.
3. Open Automated Video Analysis Platform.
4. Show the real architecture diagram.
5. Open the latest analysis result.
6. Show selected frameworks and framework rationale.
7. Show the Architecture Intelligence Score.
8. Show missing capabilities, recommendations, and trade-offs.
9. Show the agent workflow trace.
10. Show Foundry IQ grounding context and citations.
11. Open ADRs and show version history.
12. Return to Dashboard and compare Custom Document Processing Platform.
13. Explain the future roadmap.

## Presenter Notes

- emphasize that Azure AI Foundry is the AI integration path
- emphasize that Foundry IQ-style grounding is already visible
- emphasize that scoring is calculated by application code
- emphasize that seeded snapshots make the demo reliable without live cloud calls
- keep future identity and enterprise connectors clearly out of current scope
