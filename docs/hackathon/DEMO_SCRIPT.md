# Demo Script

## Purpose

Provide a polished 5-minute demo script for evaluators.

## Current Scope

Use the synthetic architecture below:

```text
A B2B SaaS platform with React frontend, .NET backend APIs, PostgreSQL/TiDB database, blob storage, background jobs, and enterprise customers in Europe. It currently has no API gateway, no tenant isolation strategy, no audit logging, no disaster recovery plan, limited monitoring, and unclear secrets management.
```

## Script

1. Open CoArchitect AI.
2. Create a workspace.
3. Add the synthetic architecture description.
4. Select Auto Detect frameworks.
5. Run analysis.
6. Show the agent reasoning trace.
7. Show the Architecture Intelligence Score.
8. Show missing capabilities.
9. Show trade-offs.
10. Generate an ADR.
11. Show Foundry IQ grounding explanation.
12. Explain the future roadmap.

## Presenter Notes

- emphasize that Azure AI Foundry is the AI integration path
- emphasize that Foundry IQ-style grounding is already visible
- emphasize that scoring is calculated by application code
- keep future identity and enterprise connectors clearly out of current scope
