# Implementation Plan

## Purpose

Describe the current implementation approach.

## Current Scope

The hackathon MVP was implemented in incremental phases:

1. workspace-centric simplification
2. local unauthenticated runtime
3. score and analysis flow
4. ADR persistence
5. UX shell and working surfaces
6. Foundry IQ-style retrieval layer
7. documentation formalization

## Design Decisions

- keep the product loop small and strong
- keep Azure integration cost-aware
- keep scoring deterministic
- keep docs aligned with code
