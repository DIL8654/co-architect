# OWASP ASVS Summary

Source:
OWASP Application Security Verification Standard. Use the official OWASP project as the canonical reference.

Summary:

- emphasizes authentication and credential handling
- validates session management and access control
- checks input handling and output safety
- covers cryptography and data protection
- reviews error handling and logging safety
- strengthens API and application-layer defenses

CoArchitect use:

- public web applications
- API-heavy systems
- multi-tenant or PII-sensitive platforms
- architectures involving external users or administrative access
