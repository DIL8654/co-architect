# Roadmap

## Phase 1: Current Unauthenticated MLP

- unauthenticated local-first MVP
- workspace-centric product shell
- workspaces, diagrams, comments, analysis, and ADR persistence
- mock analysis fallback
- local Azure-connected runtime option

## Phase 2: Multi-Agent Planning And Docs

- define multi-agent reasoning architecture
- document framework selection
- document trade-off balancing
- document ADR workflow
- create grounded knowledge base plan
- map product to Microsoft Reasoning Agents expectations

## Phase 3: Framework Selection UX

- context capture form
- manual framework selection
- auto-detect framework selection
- visible framework rationale
- quality attribute weighting UX

## Phase 4: Multi-Agent Orchestration Backend

- planner and specialist orchestration
- framework-specific specialist agents
- critic and final composer
- grounded prompts and tool contracts
- orchestration telemetry

## Phase 5: ADR HTML And PDF

- ADR creation and update UX
- HTML ADR preview
- PDF export pipeline
- ADR storage model

## Phase 6: Foundry Integration

- Microsoft Foundry or Microsoft Agent Framework integration
- Foundry IQ-style grounding
- evaluation-ready tool and memory contracts
- provider fallback strategy

## Phase 7: Evaluations And Telemetry

- reasoning quality rubric
- framework alignment evaluation
- critic effectiveness checks
- run telemetry and orchestration traces
- demo dashboards for explainability
