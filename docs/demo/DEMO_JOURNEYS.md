# Demo Journeys

## Purpose

Provide evaluator-ready journeys through the seeded CoArchitect AI demo.

## Journey 1: Explore Existing Demo Analysis

Open Dashboard  
-> choose Automated Video Analysis Platform  
-> open the diagram  
-> inspect Architecture Intelligence Score  
-> inspect Findings  
-> open Agent Workflow  
-> inspect Foundry IQ context  
-> open related ADR  
-> review ADR version history

## Journey 2: Compare Architecture Scenarios

Open Dashboard  
-> compare Automated Video Analysis Platform score  
-> compare Custom Document Processing Platform score  
-> inspect top findings and trade-offs  
-> explain why each scenario activates different reasoning concerns

## Journey 3: ADR Version History

Open Custom Document Processing Platform  
-> open ADRs  
-> select Human Review for Low-Confidence Document Extraction  
-> review v1, v2, and v3  
-> explain how the decision evolved from basic human review to thresholds and auditability

## Journey 4: Run New Analysis

Create workspace  
-> upload a diagram or create a text architecture description  
-> run AI analysis  
-> watch the agent workflow  
-> inspect findings, recommendations, trade-offs, and generated ADR output

## Current Scope

The seeded journeys are designed to be completed in under 5 minutes without Azure credentials.

## Future Enhancements

Future demos may include live Azure Foundry calls, richer comparison views, and PDF export for ADRs.
